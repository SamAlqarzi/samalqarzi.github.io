Project 4: Household Energy Consumptions

April 30, 2021 2-minute read

Project Objective:

The goal of this project is to test the factors that influence the energy consumption to fluctuate over a particular period of time. Mostly, we are examining how time itself impact power consumption, when do people tend to use more/less power. With such information, we could raise some awareness amongst consumers to help them manage their power usage.

Dataset:

We acquired our that dataset from UCI Machine Learning Repository at Dataset. This dataset was collected in France which tracked one household power usage on minutely bases for about four years.

Data Preparation

For our data preparation stage, after checking and dealing with null values and duplicated records, we decided that the dataset would produce meaningful insights if it was accumulated on daily bases. Therefore, from the beginning, we narrowed the 2 million records dataset to about 2k records by totaling the energy consumption columns.

Modeling

We built many various model to fit the data on including linear regressors, Classifiers, Trees using Rstudio and Python Facebook Prophet time series model. For more in depth on our models and their results, please see our reports as well as our code files.
