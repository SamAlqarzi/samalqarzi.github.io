Project 3: Microsoft Data For Malware Detection

April 28, 2021 2-minute read

Project Objective:

The goal of the project is to analyze Microsoft Data which was collected on windows operating devices, to to test for the probability that device might be infected by malware based on various properties for each machine.

Dataset:

Me and my team obtained our dataset from Kaggle. Each record in the dataset corresponds to a single windows operating machine and contains a huge list of properties for that machine. There are two files to this dataset, one is the train which enfolds all properties for each device and an additional column which classifies whether that particular machine is infected or not ( 1 for infected and 0 for not infected). The objective of the train set is to train the model and validate it with a small batch of that data and then use the model to predict the test set which doesn’t have that classification.

Data Preparation

Both datasets are incredibly huge that we encountered so many challenges to operate on them. For our cleaning and visualization we only operated on 1 million records for each dataset. Manny of the columns were full of missing values which we ended up excluding all columns that had almost noted every record null. Another challenge was the format of the columns, the majority of the variables were objects and categorical but there were the ones that came as float with various scales which contained version and serial numbers of each machine. So, we dealt with such columns as categorical since there are none mathematical operations to be performed on them.

Modeling

For our modeling, we faced some obstacles fitting the whole 1 million train dataset in our RandomForest Classifier model which took a considerable amount of time to complete the process, therefore we had to narrow the dataset to only a quarter of the dataset. Even for the test set we only used a small batch to test the model. For our second model(Logistic Regression Classifier), we used the whole 1 million records dataset to train and then we used the test set to predict malware detection for all machines in the dataset.
